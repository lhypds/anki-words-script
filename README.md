
Anki Words Script
=================


Features
--------

Loop all words file, generate word list form the 'Words'  
Auto import  
Auto sync  
One click to run  


Dependencies
------------

1. AnkiConnect
2. AutoHotKey
3. Python, I'm using 3.10
4. Windows batch file
*5. Windows task scheduler(optional)


How To Use
----------

Modify the path  
Run "anki_import.ahk"
